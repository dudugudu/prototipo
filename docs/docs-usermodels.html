Classe Usuarios

Esta classe representa um usuário e armazena suas informações básicas.

Propriedades

$login

Tipo: string

Login do usuário.

$nome

Tipo: string

Nome completo do usuário.

$email

Tipo: string

Endereço de email do usuário.

$celular

Tipo: string

Número de celular do usuário.

$status

Tipo: bool

Status do usuário (ativo/inativo).

Métodos

addUsuario()

Assinatura:
public void addUsuario(string $login, string $nome, string $email, string $celular, bool $status)

Carrega os dados do usuário. Este método atribui os valores fornecidos às propriedades do usuário.

Parâmetros:

Nome Tipo Descrição
$login string Login do usuário.
$nome string Nome completo do usuário.
$email string Endereço de email do usuário.
$celular string Número de celular do usuário.
$status bool Status ativo/inativo do usuário.
